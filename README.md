# GhostNet

GhostNet is a Python-based network discovery tool designed to identify active hosts within a local network. This project was created for educational purposes and to help students, hobbyists, and aspiring network engineers better understand networking concepts, host discovery techniques, and Python programming.

## Educational Purpose

This project was inspired by educational cybersecurity and networking content. The goal is to provide a learning resource for understanding how network discovery tools operate and how devices communicate within a network environment.
Feel free to study, modify, improve, and reuse the source code for your own educational projects.
